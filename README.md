## :pencil2: Overview

## :hammer: Requirement

- macOS
- Python3

## :pushpin: Usage

Git clone
```
$ git clone https://github.com/shumatsumoto/python-minigame.git
```
Move to directory
```
$ cd python-minigame
```
Virtual environment construction
```
$ python3 -m venv [newenvname]
```
Activate
```
$ source [newenvname]/bin/activate
```
Package install
```
$ pip install pygame
```
Start up PyGame
```
$ Python3 [filename]
```
